CSS-Animationen

Eine Animation erstellen

Mit einer @keyframes-Anweisung kannst Du Schritte definieren, in denen sich bestimmte CSS-Eigenschaften der Elemente verändern sollen. Im folgenden Beispiel ist eine Animation mit dem Namen "beispiel1" definiert, welche die Hintergrundfarbe von blau nach gelb verändert. Der Name kann frei gewählt werden.

CSS

          
            @keyframes beispiel1 {
              from {
                background-color: blue;
              }
              to {
                background-color: green;
              }
            }

Die @keyframes-Anweisung legt also fest, welche Eigenschaften wie verändert werden sollen. Um diese Animation dann zu verwenden, muss festgelegt werden, auf welche HTML-Elemente sie angewendet werden soll, wie schnell sie ablaufen soll, und wie oft sie wiederholt werden soll. Das folgenden Beispiel zeigt, wie die gerade definierte Animation "beispiel1" auf ein HTML-Element angewendet werden kann.

CSS

          
            div.animiert {
              width:400px;
              height: 400px;
              /* Die Animation wird anhand ihres Namens ausgewählt */
              animation-name: beispiel1;
              /* Die Dauer der Animation wird festgelegt */
              animation-duration: 1.5s;
              /* Es kann eine Verzögerung für den Beginn festgelegt werden */
              animation-delay: 0s;
              /* Wie oft soll die Animation wiederholt werden?
                Es kann eine Zahl angegeben werden oder "infinite", dann läuft sie dauerhaft.
              */
              animation-iteration-count: infinite;
              /* Soll die Animation von Anfang bis Ende laufen (normal),
              oder rückwärts (reverse), oder immer vor und zurück (alternate),
              oder zurück und vor (alternate-reverse?)? */
              animation-direction: alternate;

            }

            /* Wenn die Maus über dem div ist, soll die Animation pausieren. */
            div.animiert:hover {
              animation-play-state: paused;
            }

Schritte zu einer Animation hinzufügen

In der oben definierten @keyframe-Anweisung sind zwei Schritte festgelegt: from und to. Es gibt aber auch die Möglichkeit, mehrere Schritte anzugeben. Anstatt from und to verwendet man dann einfach Prozentwerte um zu definieren, wann die Veränderung passieren soll. Soll etwas beispielsweise genau nach der Hälfte der Animation passieren, dann nimmt man 50% als Wert. Im folgenden Beispiel findet jeweils nach 20% die nächste Veränderung statt.

CSS

          
            @keyframes beispiel2 {
              0% {
                background-color: blue;
              }
              33% {
                width: 30px;
                height: 30px;
              }
              66% {
                width: 120px;
                height: 60px;
              }
              100% {
                background-color: green;
              }
            }
