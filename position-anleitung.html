Die Eigenschaft "position"

Alle HTML-Elemente werden standardmäßig in den Seitenfluss eingebettet. Das heisst sie werden an der Position angezeigt, an der sie im Quellcode definiert sind.

Mit der CSS-Eigenschaft "position" können Elemente auf der Seite anders positioniert werden. Die Eigenschaft kann folgende Werte haben:

  • relative: Das Element wird relativ zu seiner eigenen Position verschoben. Die Verschiebung wird über die Eigenschaften top, left, bottom und right angegeben.
  • absolute: Das Element wird aus dem Seitenfluss genommen und mit top, left, bottom und right frei auf der Seite positioniert. Wenn ein Elternelement relativ positioniert ist, dann wird dieses Elternelement als Bezugsrahmen für die Verschiebung genommen.
  • fixed: Das Element wird wie bei position: absolute aus dem Seitenfluss genommen, bleibt aber auf der Seite liegen und scrollt nicht mit.
  • sticky: Das Element scrollt solange mit, bis es die festgelegte Position erreicht hat, und bleibt dann dort kleben.
  • static: Das ist die Standardeinstellung und dasselbe, als wenn nichts angegeben wird. Das Element wird einfach im Seitenfluss angezeigt.

CSS

        

          /* ohne besondere Positionierung. In dem Fall wird das Element
          statisch positioniert. Keine Angabe entspricht also position: static */


          div {
            border: 1px solid black;
            width: 50px;
            height: 50px;
          }

          /* relativ positioniert: Der Platz des Element auf der Seite
          bleibt erhalten, es wird relativ von dort verschoben */

          .second {
            position: relative;
            top: -10px;
            left: 10px;
          }

          /* absolut positioniert: Das Element wird aus dem Seitenfluss
          genommen und frei auf der Seite positioniert. */

          .third {
            position: absolute;
            bottom: 10px;
            right: 10px;
          }


          /* "sticky": Das Element bleibt am Seitenanfang "kleben",
          wenn weiter nach unten gescrollt wird. */

          .fourth {
            position: sticky;
            bottom: 10px;
            right: 10px;
          }

          /*Ist das Eltern-Element nicht statisch positioniert
          (also z.B. relativ), dann... */

          .outer {
            position: relative;
            width: 100px;
            height: 100px;
            background-color: #0000aa;
          }

          /*... orientiert sich das absolut positionierte Kind-Element
          am Eltern-Element, und nicht mehr an der gesamten Seite.*/

          .inner {
            position: absolute;
            top: 10px;
            left: 20px;
            background-color: #00aa00;
          }
