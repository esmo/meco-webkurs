CSS-Transformationen

Was sind Transformationen?

Mit CSS-Transformationen werden HTML-Elemente mit Hilfe von CSS verschoben, gedreht, skaliert oder verzerrt. Das folgende Beispiel zeigt diese vier Transformations-Möglichkeiten anhand von 4 Quadraten.

CSS

          
            .skew {
              /* Die Stärke der Verzerrung wird im Bogenmaß (0-360 Grad) angegeben,
              jeweils für die Höhe und die Breite. */
              transform:skew(10deg,20deg)
            }

            .translate {
              /* Verschiebung von nach rechts und nach unten,
              negative Werte verschieben nach links bzw. nach oben. */
              transform:translate(-20px, 30px);
            }

            .rotate {
              /* Die Rotierung wird auch im Bogenmaß angegeben. */
              transform:rotate(50deg)
            }

            .scale {
              /* Die beiden Werte bei der Skalierung stehen für horizontal und vertikal.
              Ein Wert von 1 entspricht der Originalgröße */
              transform:scale(0.5,2)
            }
