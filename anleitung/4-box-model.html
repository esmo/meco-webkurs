Das CSS-Box-Model

Die Bestandteile eines HTML-Elements

Jedes HTML-Element besteht im Wesentlichen aus einer Box, die einen Aussenabstand (margin), einen Rand (border), einen Innenabstand (padding) und einen Inhaltsbereich (content) hat. Das folgende Bild veranschaulicht diese Bereiche:

Zusammengenommen wird dies als "Box-Model" bezeichnet. Mit den folgenden CSS-Anweisungen können die einzelnen Elemente des Box-Models gestaltet werden: 

        
          .box {
            /* Der Aussenabstand wird auf 100 Pixel gesetzt */
            margin: 100px;
            /* Der Innenabstand wird auch auf 100 Pixel gesetzt */
            padding: 100px;

            /* Der Rahmen soll 50 Pixel breit sein */
            border-width: 50px;
            /* Der Rahmen soll eine einfache Fläche sein.
            Es gibt auch andere Optionen, wie bspw. eine
            gestrichelte Linie. */
            border-style: solid;
            /* Die Rahmenfarbe wird auf schwarz gesetzt */
            border-color: black;
            /*Die Hintergrundfarbe des Inhaltsbereichs wird auf grün gesetzt.*/
            background-color: green;

            /* Die Höhe des Inhaltsbereichs soll 100 Pixel betragen */
            height: 100px;
            /* Die Breite des Inhaltsbereichs soll auch 100 Pixel betragen */
            width: 100px;

          }

Angaben für oben, unten, links und rechts

Es ist möglich, für den Innenabstand, den Rahmen und den Aussenabstand unterschiedliche Angaben für die einzelnen Seiten zu machen. Dann werden die gewünschten Werte einfach hintereinander angegeben, wie in den folgenden Beispielen. 

        
          /* Bei vier Werten sind diese für oben, rechts, unten und links */
          margin: 10px 2px 2px 5px;
          padding: 10px 2px 2px 5px;
          border-width: 10px 2px 2px 5px;
          /* Bei drei Werten ist der erste für oben, der zweite für
          rechts und links, und der dritte für unten */
          margin: 10px 2px 5px;
          padding: 10px 2px 5px;
          border-width: 10px 2px 5px;
          /* Bei zwei Werten ist der erste für oben und unten, und der zweite für
          rechts und links  */
          margin: 10px 2px;
          padding: 10px 2px;
          border-width: 10px 2px;
          /* Bei einem Wert gilt dieser für alle Seiten, wie schon oben beschrieben. */
          margin: 10px;
          padding: 10px;
          border-width: 10px;

Angaben nur für bestimmte Seiten

Es ist auch möglich, die Angaben nur für bestimmte Seiten zu machen. Will man beispielsweise nur den linken Aussenabstand auf 10 Pixel setzen, so kann man dies mit der Angabe margin-left: 10pxerreichen. Die Angaben für die einzelnen Seiten sind wie folgt. 

        
          .box {
            /* Der Aussenabstand wird auf 100 Pixel gesetzt */
            /* oben */
            margin-top: 100px;
            /* rechts */
            margin-right: 100px;
            /* unten */
            margin-bottom: 100px;
            /* links */
            margin-left: 100px;

            /* Der Innenabstand wird auch auf 100 Pixel gesetzt */
            /* oben */
            padding-top: 100px;
            /* rechts */
            padding-right: 100px;
            /* unten */
            padding-bottom: 100px;
            /* links */
            padding-left: 100px;

            /* Der Rahmen soll 50 Pixel breit sein */
            /* oben */
            border-top-width: 50px;
            /* rechts */
            border-right-width: 50px;
            /* unten */
            border-bottom-width: 50px;
            /* links */
            border-left-width: 50px;
          }

Zusammenfassende Angaben

Wichtig: Die Eigenschaften width und height eines HTML-Elements legen nur die Breite und Höhe des Inhaltsbereichs fest. Um die volle Größe eines Elements zu berechnen, müssen auch Aussenabstand (margin), Rand (border) und Innenabstand (padding) berücksichtigt werden.
Mit der CSS-Eigenschaft box-sizing: border-box; kann das Verhalten aber so verändert werden, dass die Werte für Höhe und Breite Innenabstand und Rand mit einschließen.
