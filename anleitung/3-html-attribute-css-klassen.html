HTML-Attribute und CSS-Selektoren

HTML-Elemente können Attribute haben. Attribute stellen zusätzliche Informationen zu Elementen bereit. Sie werden immer öffnenden Teil des HTML-Tags angegeben. Ein Beispiel für die Attribute class und id in einer Überschrift könnte so aussehen: 

        
          

            Beispiel-Überschrift

Die Attribute class und id haben eine spezielle Bedeutung. Sie können in CSS als Selektoren verwendet werden und haben dort eine besondere Schreibweise:

  • Für das class-Attribut wird einfach der gewünschte Wert mit einem Punkt davor geschrieben, also bspw. .beispiel-klasse
  • Für das id-Attribut wird der gewünschte Wert mit einem Hash-Zeichen (#) davor geschrieben, also bspw. #beispiel-id

Die beiden Attribute class und id unterschieden sich in einigen Punkten.

  • Ein class-Attribut kann mehrere Werte haben . Diese werden dann einfach mit Leerzeichen getrennt hintereinander geschrieben. Die Werte können bei mehreren HTML-Elementen auf einer Seite verwendet werden.
  • Anders ist es bei id-Attributen: Ein id-Attribut darf nur einen Wert haben, und dieser Wert darf nur einmal in dem HTML-Dokument vorkommen.

Ein id-Attribut dient also dazu, ein HTML-Element in einem HTML-Dokument eindeutig zu identifizieren. Eine class-Attribut hingegen dient dazu, mehrere HTML-Elemente zu gruppieren, also zu einer Klasse zusammenzufassen.

CSS

        

          /* Mit der folgenden CSS-Anweisung wird die Schriftfarbe für die
          HTML-Elemente mit der Klasse "beispiel-klasse" blau */
          .beispiel-klasse {
            color: blue;
          }

          /* Mit der folgenden CSS-Anweisung wird die Schriftfarbe für das
          HTML-Element mit der ID "beispiel-id" grün */
          #beispiel-id {
            color: green;
          }
