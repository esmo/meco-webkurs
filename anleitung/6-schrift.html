Textgestaltung mit CSS

Für die Textgestaltung stehen in CSS eine Reihe von Möglichkeiten zur Verfügung. Im folgenden sind die wichtigsten aufgeführt. Ein Beispieldokument mit allen möglichen CSS-Eigenschaften zur Schriftgestaltung findet sich hier: Textgestaltung mit CSS

Schriften einbinden

        

          /* Import einer Schriftart, bspw. von Google Fonts */
          @import url('https://fonts.googleapis.com/css2?family=Roboto:wght@100&display=swap');
          /* Verwenden einer bestimmten Schriftart und einer Ersatzschriftart,
          falls die Schrift nicht geladen werden kann. */
          font-family: Roboto, sans-serif;

Schriftgröße und Zeilenabstand

Die Schriftgröße kann über die Eigenschaft font-size festgelegt werden.

Die Festlegung der Zeilenhöhe hat eine Besonderheit: Es ist möglich, sie in Abhängigkeit von der Schriftgröße zu definieren. Dafür gibt man einfach einen Wert ohne Einheit an, also bspw. 1. Der Wert dient als Multiplikator, d.h. die Zeilenhöhe entspricht der Schriftgröße multipliziert mit diesem Wert .

  • Ist der Wert 1, dann entspricht die Zeilenhöhe der Schriftgröße.
  • Ist der Wert zwischen 0 und 1, dann ist die Zeilenhöhe kleiner als die Schriftgröße.
  • Ist der Wert größer als 1 , dann ist die Zeilenhöhe um diesen Faktor größer als die Schriftgrößer 
        

          /* Schriftgröße festlegen */
          font-size: 12px;

          /* Zeilenhöhe festlegen. In diesem Fall ist die Zeilenhöhe das
          1,5-fache der Schriftgröße, also 18 Pixel. */
          line-height: 1.5;

Das Schriftbild verändern

        

          /* fett */
          font-weight: bold;

          /* kursiv */
          font-style: italic;

          /* Schriftfarbe */
          color: blue;

          /* Unter-, Über-, und Durchstreichungen */
          text-decoration: underline;
          text-decoration: overline;
          text-decoration: line-through;
          /* Verhindert bspw. die automatische Unterstreichung von
          Links (a-Tags). */
          text-decoration: none;


          /* Klein- bzw. Großbuchstaben erzwingen */
          text-transform: lowercase;
          text-transform: uppercase;
          /* Kapitälchen */
          text-transform: capitalize;

          /* selbstdefinierte Anführungszeichen verwenden */
          quotes: '„' '”';
