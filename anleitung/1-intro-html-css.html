HTML und CSS

Eine Einführung

HTML und CSS sind zwei verschiedene Programmiersprachen, die aber zusammenarbeiten.

HTML

Mit HTML werden Inhalte in einem Dokument strukturiert. Beispielsweise wird festgelegt, welcher Text die Überschrift sein soll und was zu einem Text-Absatz gehören soll. Ausserdem werden Medien - wie Bilder und Videos - in das Dokument eingebunden.

HTML-Elemente werden mithilfe von sogenannten Tags markiert. Der Inhalt wird dafür zwischen dem öffnenden und dem schließenden Teil des HTML-Tags eingeklammert. HTML-Elemente können dabei verschachtelt werden, d.h. ein HTML-Element kann weitere HTML-ELemente enthalten.

CSS

Mit CSS werden die HTML-Elemente gestaltet. Beispielsweise kann die Größe, Farbe und Schriftart der Überschrift festgelegt werden, oder der Abstand des Text-Absatzes nach oben und unten.

Dafür wird in der CSS-Anweisung das HTML-Element ausgewählt, das gestaltet werden soll. Das geschieht mit dem sogenannten Selektor (selektieren = auswählen). Dann werden ihm die gewünschten Eigenschaften zugewiesen. Der Selektor wird an den Anfang der CSS-Anweisung geschrieben. Die Eigenschaften werden dahinter in geschweifte Klammern geschrieben. 

        
          /* Dies ist ein Beispiel für einen Kommentar in CSS.*/

          /* Mit der folgenden CSS-Anweisung wird die Schriftfarbe für die Überschrift (h1) blau */
          h1 {
            color: blue;
          }

          /* Mit der folgenden CSS-Anweisung wird die Schriftfarbe für den Text-Absatz (p) rot */
          p {
            color: red;
          }
