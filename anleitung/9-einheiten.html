Maßeinheiten in CSS

Es gibt verschiedene Maßeinheiten, die in CSS verwendet werden können, um die Größe von Elementen festzulegen. Dabei wird unterschieden zwischen absoluten und relativen Maßeinheiten..

  • absolute Maßeinheiten legen die Größe eindeutig fest, unabhängig davon, wie groß andere Elemente sind.
  • relative Maßeinheiten legen die Größe in Abhängigkeit vom jeweiligen Elternelement fest.

Pixel (Einheit px) sind die wichtigste absolute Maßeinheit . Ein Pixel ist ein Punkt auf dem Bildschirm. Je nachdem, wie hoch die Auflösung des Bildschirms ist, kann ein Pixel jedoch unterschiedlich groß sein. Je höher die Auflösung ist, desto kleiner sind die Pixel.

Die einfachste und bekannteste relative Maßeinheit sind Prozentangaben. Diese Angabe bezieht sich immer auf den Wert des Elternelement. (Mit einer seltsamen Ausnahme: Die Prozentangabe beim Aussenabstand margin bezieht sich immer auf die Breite des Elternelements.)

Die Einheit em ist auch eine relative Angabe, sie entspricht aber der an dieser Stelle verwendeten Schrift. Das ist überall da sinnvoll, wo Abstände in Abhängigkeit von der Schriftgröße festgelegt werden sollen.

Die Einheiten vw und vh beziehen sich auf das Anzeigefenster (engl. "viewport") des Browsers. 100vh sind 100% der Höhe (vh = viewport height) und 100vw 100% der Breite (vw = viewport width) des Anzeigefensters.

Im folgenden Beispiel kommen alle angesprochenen Maßangaben vor.

        

          div {
            /* Der Rand wird 1 Pixel breit gesetzt. */
            border: 1px solid black;

            /* Der Innenabstand wird auf die
              doppelte Größe der Schrift gesetzt. */
            padding: 2em;

            /* Die Schriftgröße wird auf 80% der Schriftgröße
              des Elternelements gesetzt. */
            font-size: 80%;

            /*Die Breite wird auf 1% der Breite
            des Anzeigefensters gesetzt. */
            width: 1vw;

            /*Die Höhe wird auf ein Viertel (25%)
              der Höhe des Anzeigefensters gesetzt */
            height: 25vh;
          }
