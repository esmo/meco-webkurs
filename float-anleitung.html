Float

Umfließender Text

Mit der Eigenschaft float können Elemente so gestaltet werden, dass sie von anderen Elementen "umflossen" werden. So kann beispielsweise Fließtext um Bilder oder andere Elemente realisiert werden.

Mit float: left wird festgelegt, dass das Element links vom umfließenden Inhalt steht, also rechts umflossen wird. Entsprechendes gilt für float: right

Es kann notwendig sein, das Umfließen für ein spezifisches Element aufzuheben. Das geschieht mit der Eigenschaft clear: left, clear: right, oder clear: both.

CSS

        
          /* Angabe für ein links gefloatetes Element */
          .left-floated {
            float: left;
          }

          /* Angabe für ein rechts gefloatetes Element */
          .right-floated {
            float: right;
          }

          /* Textgröße ändern, damit der Text über mehrere Zeilen läuft.  */
          .bigtext {
            font-size: 4em;
            line-height: 1em;
          }
