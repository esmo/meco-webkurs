Flexbox

Elemente positionieren

Mit Hilfe der Flexbox kann man HTML-Elemente horizontal und vertikal positionieren. So können einzelne Elemente beispielsweise zentriert werden, oder mehrere Elemente nebeneinander oder untereinander dargestellt werden. Das ist eine einfache und flexible Möglichkeit, Layouts zu erstellen.

Um Elemente zu positionieren, müssen sie in einem sogenannten Flex-Container sein. Einen Flex-Container bekommt man, wenn man einem HTML-Element die CSS-Anweisung display:flex; gibt. Damit werden automatisch alle direkten Kindelemente dieses Elements zu sogenannten Flex-Items, die über weitere CSS-Regeln positioniert werden können.

Das einfachste Flexbox-Beispiel sieht folgendermaßen aus:

      
      .flex-container {
        display: flex;
      }

Gibt man einem HTML-Element die Klasse flex-container, dann werden also alle direkt darunter liegenden Elemente als Flex-Items behandelt.

Aber alleine ist diese CSS-Anweisung wirkungslos, sie bekommt erst im Zusammenspiel mit anderen Anweisungen einen Sinn. Mit der CSS-Eigenschaft flex-direction lässt sich beispielsweise bestimmen, dass die Flex-Items vertikal oder horizontal angeordnet werden sollen. Im folgenden Beispiel wollen wir die Elemente untereinander - in eine Spalte (column) - stellen. 

    
    .flex-container {
      display: flex;
      flex-direction: column; /* Die Elemente werden untereinander dargestellt */
    }

Wird der Wert für flex-direction auf row geändert, dann werden die Flex-Items nebeneinander - in einer Reihe (row) - dargestellt. 

    
      .flex-container {
        display: flex;
        flex-direction: row; /* Die Elemente werden nebeneinander dargestellt */
      }

Über die Eigenschaft justify-content kann festgelegt werden, wie die Elemente zueinander angeordnet werden. Der Wert center beispielsweise zentriert alle Elemente, und mit space-around werden alle Elemente mit dem gleichen Abstand zueinander (und zum Rand) dargestellt. Ist nichts angegeben, werden die Elemente linksbündig angeordet, was dem Wert flex-start entspricht. 

    
      .flex-container {
        display: flex;
        flex-direction: row; /* Die Elemente werden nebeneinander dargestellt */
        justify-content: space-around; /* Alle Elemente haben den gleichen Abstand zueinander und zum Rand. */
      }
